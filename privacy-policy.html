Privacy Policy

OTB URL Copier  ·  Last updated May 2026

Overview

OTB URL Copier is a Chrome extension that adds a one-click copy button to event bars on Gradient Sports. When clicked, it copies a transformed version of the event URL to your clipboard.

This extension is operated privately and is not affiliated with Gradient Sports.

Data Collection

This extension collects no personal data whatsoever.

What it does: Reads the href attribute from event bar links on soccer-dat.gradientsports.com and writes a transformed URL to your clipboard.

What it does not do: No data is collected, stored, logged, transmitted, or shared — with us or any third party. Nothing ever leaves your device.

Permissions

The extension requests the following permissions:

Host access to soccer-dat.gradientsports.com
Required to inject the copy button into the page UI and read event link URLs.
Clipboard write
Required to copy the transformed URL to your clipboard when you click the button.

Third Parties

This extension does not use any analytics, tracking, advertising, or third-party services of any kind.

Contact

If you have any questions about this privacy policy, please open an issue on the extension's GitHub repository or contact the developer directly.
